# Kolam

A simple kolam drawing game. Connect the dots to create patterns.

## Run

Open `index.html` in a browser (double-click or run a local server).

```bash
# optional: serve locally
npx serve .
# or
python3 -m http.server 8000
```

Then visit http://localhost:3000 (or 8000).

## How to play

- **Draw:** Click and drag from one dot to another to draw a line.
- **Clear:** Click "Clear" to remove all lines and start over.

No build step—just HTML, CSS, and JavaScript.
